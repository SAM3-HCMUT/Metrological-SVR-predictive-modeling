# Metrological-SVR-predictive-modeling
Python codebase for SVR-RBF surrogate modeling, SHAP-based explainable AI (XAI) analysis, and rigorous ISO-GUM quantitative uncertainty propagation. Features Leave-One-Out CV, Wilcoxon error testing, and journal-standard error-bar plots for V-grooving assisted V-bending process prediction. Designed for full reproducibility.
